Thoughtful Solitaire Domain:
----------------------------

This domain models the game of thoughtful solitaire, which is a solitaire
variant that is played with all of the cards visible. 

The problem difficulty is scaled by varying the number of cards and 
stacks. 

The competition problems have all been proven to be solvable using the 
solver described in:

Searching Solitaire in Real Time
Ronald Bjarnason, Prasad Tadepalli, and Alan Fern
International Computer Games Association Journal, 30 (3), 131-142, 2007
http://www.cs.orst.edu/~afern/papers/solitaire.pdf

A more detailed description of the game can be found in the above
paper.
